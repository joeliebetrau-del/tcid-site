Inventory Mapper

How the mapper works.

The mapper handles real-world inventory data in any format -- from single-cell descriptions to multi-column spreadsheets with inconsistent headers.

01

Paste or Upload

Paste card descriptions one per line, or upload an Excel or CSV file. Select which columns to include -- the mapper handles the rest.

02

AI Parsing

Each row is parsed by AI into structured fields -- player, year, manufacturer, product, parallel, grade, and more -- regardless of input format or column naming.

03

Registry Match

Parsed fields are searched against the TCID registry. Each card returns a matched TCID and confidence score, or a clear "not in registry" flag.

04

Export Results

Download the enriched inventory as .xlsx, or copy tab-separated data directly into Google Sheets with one click.
